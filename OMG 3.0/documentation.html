2. Learn to design web pages

You can start here:

Hello, world! This is the first page for you.


<!doctype html>
<html lang="en">
<head>
<meta charset="utf-8">
<!-- This is the Bootstrap core CSS and JS -->
<link rel="stylesheet" href="bootstrap-5.1.3-dist/css/bootstrap.css">
<script src="bootstrap-5.1.3-dist/js/bootstrap.js"></script>
<!-- Custom style -->
<style>
body {
font-family: "Avenir Next";
}
</style>
</head>
<body>
<h1>Hello, world!</h1>
</body>
</html>

And, you can add a button for sure:

Hello, world!
